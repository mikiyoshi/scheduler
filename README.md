# Interview Scheduler

## Setup

Install dependencies with `npm install`.

### macOS Monterey

```
nvm install 12.22.7
scheduler % node -v
v12.22.7
scheduler % nvm -v
0.37.2
scheduler % npm -v
8.3.1
```

## Running Webpack Development Server port 8000

- scheduler localhost:8000
  - package.json proxy localhost:8001
- scheduler-api localhost:8001

## scheduler and scheduler-api run

```sh
npm start
```
